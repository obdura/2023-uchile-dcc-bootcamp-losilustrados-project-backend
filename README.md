# **Los Ilustrados** - Backend

## Descripción

Proyecto backend - equipo *Los Ilustrados*
## Descarga del repositorio

```bash
$ git clone https://github.com/obdura/2023-uchile-dcc-bootcamp-losilustrados-project-backend.git

```

## Instalación

```bash
$ npm install
```

## Ejecutando la app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
